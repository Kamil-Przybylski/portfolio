O mnie

Nazywam się Kamil Przybylski, mieszkam w Krakowie, mam 25 lat, z wykształcenia jestem projektantem konstrukcji budowlanych.

Od grudnia przebranżawiam się na Front-End Developer'a, naukę prowadzę we własnym zakresie z kursów internetowych. Aktualnie szukam pracy na stanowisko Junior Front-End Developer'a. Jeżeli poszukujesz takiej osoby to skontaktuj się ze mną!

~ kprzybylski993@gmail.com

O mnie - więcej

Wykształcenie

Studiowałem na Politechnice Krakowskiej im. Tadeusza Kościuszki gdzie uzyskałem tytuł magistra inżyniera na kierunku Budownictwo na wydziale Inżynierii Lądowej.

Znam angielski na poziomie średnim, pozwalającym czytać oraz szukać rozwiązań związanych z programomwaniem po angielsku.

Zawód

Po studiach rozpocząłem prace jako asystent projektanta konstrukcji w biurze projektowym KB-Projekty Konstrukcyjne. Zajmowałem się obliczeniami statycznymi konstrukcji, rysunkami budowlanymi i wykonawczymi oraz ekspertyzami.

Po 9 miesiącach pracy zwolniłem się aby więcej czasu poświęcić nauce programowania.

Programowanie

Programowania uczę się we własnym zakresie z kursów internetowych. Posiadam wiedzę z HTML5, CSS3, JS w ES6 oraz dodatkowo jQuery, Vue.js, SASS, Bootstrap, webpack, Gulp, Git.

Ukończyłem kursy:

  • JavaScript - od Podstaw ~ eduweb.pl
  • JavaScript - w Praktyce ~ eduweb.pl
  • Kurs HTML5 i JavaScript - Techniki Zaawansowane ~ eduweb.pl
  • Kurs Programowania w jQuery - w Praktyce ~ eduweb.pl
  • Kurs Vue.js - od Podstaw ~ eduweb.pl
  • Kurs Tworzenia Stron WWW w HTML i CSS od podstaw do eksperta ~ udemy.com

Technologie

HTML5

HTML5

W projektach wykorzystuję nowe znaczniki semantyczne w HTML5. Potrafię wykorzystywać HTML z CSS i JS do tworzenia responsywnych stron.

CSS3

CSS3

Znam podstawowe oraz zaawansowane właściwości, pseudoelementy, pseudoklasy, selektory i pseudoselektory. Wykorzystuję flex-box'a oraz potrafię tworzyć animacje.

JavaScript

JavaScript

Rozumiem programowanie obiektowe, działanie obiektowego modelu domu oraz AJAX'a i JSON'a. Ukończyłem kursy związane z jQuery i Vue.js. Znam najnowszą składnię ES6.

jQuery

jQuery

Ukończyłem internetowy "Kurs Programowania w jQuery - w Praktyce" z eduweb.pl związany z wyszukiwaniem elementów, zdarzeniami, stylami, animacjami oraz AJAX'em itp.

Vue.js

Vue.js

Ukończyłem internetowy "Kurs Vue.js - od Podstaw" z eduweb.pl przedstawiający sposób pracy z Vue, zdarzeniami, pracą ze stylami, filtrami oraz tworzenia komponentów i animacji.

SASS

SASS

Wiedzę na temat SASS zdobyłem z "Kursu tworzenia stron www od podstaw do eksperta" z udemy.com w którym przedstawione były m.in. własne zmienne, mixiny, funkcje, pętle itp.

Bootstrap

Bootstrap

Naukę z Bootstrapem zacząłem na youtube.com, resztę wiedzy zdobyłem przy tworzeniu własnej strony czytając dokumentację framework'a.

Git

Git

Większość projektów przechowuję na github.com gdzie wykorzystuję darmowy hosting do stron internetowych. Do aktualizowania projektów korzystam z "GitHub Descopt" oraz plugin'ów w Visual Studio Code.

Visual Studio Code

Visual Studio Code

Wszystkie aplikacje piszę w programie Visual Studio Code. Wykorzystuję dodatkowe plugin'y przyśpieszające pracę typu Emmet, Snippets, Live Server, Live SASS Compiler, Debugger for Chrome itp.

Gulp

Gulp

Poznałem Gulp'a przy okazji "Kursu tworzenia stron www od podstaw do eksperta" z udemy.com.

Webpack

Webpack

Poznałem Webpack'a przy okazji "Kursu Vue.js - od Podstaw" z eduweb.pl gdzie wykorzystywany był do kompilowania plików z rozszerzeniem .vue.
